About

Block Runner is a 3-D game powered by the Unity Engine where you are controlling a block that has to dodge obstacles. The game is infinite and your score is based on how far you have traveled. There are three difficulties: easy, medium, and hard. Block Runner is currently available for Windows and on browsers supporting WebGL. The android version is still work in progress and may not be working. Controllers are supported.

Download

Block Runner is also available to be played online in your browser using WebGL.
